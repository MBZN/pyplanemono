PIP Installation
----------------

To install the latest release of the package, you can use the following command:

.. code-block:: bash

    pip install pyplanemono


Source Installation
-------------------

To install the latest development version of the package, you can clone the repository and install it using the following commands:

.. code-block:: bash

    git clone https://github.com/patrickwang27/pyplanemono
    cd pyplanemono
    pip install . -e
